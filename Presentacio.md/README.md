# 📄 Presentació final



---


Aquest projecte forma part de la presentació final i té com a objectiu mostrar el desenvolupament, les funcionalitats i els aprenentatges adquirits durant el procés de creació.

Al llarg del projecte s’ha treballat en diferents àrees com el disseny, la implementació i la validació del sistema, amb l’objectiu d’aconseguir una solució funcional i ben estructurada.

En aquesta presentació s’explicarà el context del projecte, les tecnologies utilitzades i les decisions més importants que s’han pres durant el desenvolupament.
